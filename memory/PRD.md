# Mejora Tu WEB - PRD

## Problem Statement
SaaS de conversion que analiza webs con IA. Prioridad: monetizacion inmediata.

## Architecture
- Frontend: React + Tailwind + Framer Motion + Shadcn UI + jsPDF
- Backend: FastAPI + MongoDB + emergentintegrations
- AI: OpenAI GPT-4o-mini
- Auth: Emergent Google OAuth (opcional post-pago)
- Payments: Stripe Payment Link (5 EUR)
